# ParkingSmart Standard

ParkingSmart Standard is an offline smart parking management software designed for Windows systems.

## Status

| Sprint | Status | Description |
|--------|--------|-------------|
| Sprint 1 | ✅ Completed | Foundation & ALPR Architecture with Mock support |
| Sprint 2 | ⏳ Planned | YOLO-based plate detection |
| Sprint 3 | ⏳ Planned | PaddleOCR integration |
| Sprint 4 | ⏳ Planned | Parking core logic |

## Features

- ✅ Offline operation
- ✅ CPU-only execution
- ✅ Iranian license plate recognition (Mock mode for testing)
- ✅ SQLite database
- ✅ Easy installation
- ✅ High stability
- ✅ Upload image for plate testing
- ✅ Bounding box visualization
